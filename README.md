## Getting Started

### Installation

Install `capitalisation_station` with pip:

```shell
pip install capitalisation_station
```

### Setup for Development

If you're looking to contribute or develop with `capitalisation_station`, get the source code and set up the environment:

```shell
git clone https://github.com/eightballer/capitalisation_station
cd capitalisation_station
poetry install && poetry shell
```

## Commands

Here are common commands you might need while working with the project:

### Formatting

```shell
make fmt
```

### Linting


### Testing


### Locking
